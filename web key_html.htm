What is the tag responsible for defining html5?

The code


  < !DOCTYPE  html >

It is the basic code to define that the file is an HTML5 file,
All HTML documents must start with a <!DOCTYPE> declaration,
The declaration is not an HTML tag.
It is an "information " to the browser about what document type to expect.

What is the tag that contains all the tags of the html page?

The code


  < html  lang ="en" >

  < /html >

The <html> tag represents the root of an HTML document.
The <html> tag is the container for all other HTML elements (except for the <!DOCTYPE> tag).
Note: You should always include the lang attribute inside the <html> tag, to declare the language of the Web page.
This is meant to assist search engines and browsers.

lang:

The lang attribute specifies the language of the element's content.
Common examples are "en" for English, "es" for Spanish, "fr" for French.

Text tags


1-

open tag:

<body></body>

The<body> tag defines the document's body.
The <body>element contains all the contentsof an HTML document, such as headings, paragraphs,
  images, hyperlinks, tables, lists, etc.
Note:There can only be one <body> element in an HTML document.


2-

open tag:

<h1> to <h6>

The <h1> to <h6> tags are used to define HTML headings.
<h1> defines the most important heading. <h6> defines the least important heading.
Note: Only use one <h1> per page - this should represent the main heading/ subject for the
  whole page. Also, do not skip heading levels - start with <h1>, then use <h2>, and so on.


3-

open tag:

<p></p>

The <p> tag defines a paragraph.
Browsers automatically add a single blank line before and after each <p> element.
Tip: Use CSS to style paragraphs.


4-

open tag:

<span></span>

The <span> tag is an inline container used to mark up a part of a text, or a part of a document.
The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
The <span> tag is much like the <div> element, but <div> is a block-level element
and <span> is an inline element.


5-

open tag:

<font></font>

The <font>tag was used in HTML 4 to specify the font face, font size, and color of text.
<font> in HTML5. Removed due to having CSS code similar to the same function
(color: ; / font-family: ; / font-size: ;).
Click here to go to the course.


6-

closed tag:

<br>

The <br>tag inserts a single line break.
The <br> tag is useful for writing addresses or poems.
The <br> tag is an empty tag which means that it has no end tag.
Note: Use the <br> tag to enter line breaks, not to add space between paragraphs.


7-

closed tag:

<hr>

The <hr>tag defines a thematic break in an HTML page (e.g. a shift of topic).
The <hr>element is most often displayed as a horizontal rule that is used to
  separate content (or define a change) in an HTML page.


8-

open tag:

<u> and <ins>

The <u>and<ins> tags represent some non-articulated text that is styled
differently than regular text, such as misspelled words.


9-

open tag:

<s> and <del>

The <s> tag specifies text that is no longer correct, accurate or relevant.
The text will be displayed with a line through it.
The <s> tag should not be used to define deleted text in a document,use the <del> tag for that.


10-

open tag:

<b></b>

The <b>tag defines bold text without any additional significance.
Note: According to the HTML5 specification, the <b> . file
The tag should be used as a last resort when no other tag is more appropriate.
The specification states that addresses should be denoted by <h1> to <h6> tags.
Tip: You can also use the following CSS to set bold text: Line weight: 900; .


11-

open tag:

<strong></strong>

The<strong>tag is used to define text with strong importance.
   The content inside is typically displayed in bold.
Tip: Use the <b> tag to specify bold text without any extra importance!


12-

open tag:

<i></i>

The<i>tag identifies a piece of text with an alternate sound or mood.
  The internal content is usually shown in italics.
The <i> tag is often used to refer to a technical term, a phrase from another language, an idea, etc.
Use the <i> element only when there is no more appropriate semantic element, such as:
<em> (highlighted text).


13-

open tag:

<em></em>

The<em> tag is used to define emphasized text. The content inside is typically displayed in italic.


14-

open tag:

<sup></sup>

The<sup>tag specifies the superscript. Superscript appears Half a character above the regular line.


15-

open tag:

<sub></sub>

The<sub> tag specifies the subscript. The subscript appears with a half character below the regular line,
  the subscript can be used for chemical formulas, Like H2so4: sulfuric acid.
Tip: Use the <sup> tag to define superscript text.


16-

open tag:

<mark></mark>

The<mark> tag defines text that should be marked.

<center></center>

The<nav> tag defines a set of navigation links.
Notice that NOT all links of a document should be inside a <nav> element.
The <nav> element is intended only for major block of navigation links.

id:

The HTML property id is used to specify a unique identifier for an HTML element.
You cannot have more than one element with the same (identifier) ​​in an HTML document.
The attribute id is used to indicate a specific style declaration in a style sheet.
It is also used by JavaScript to access and manipulate the element with the id specified.

1-

open tag:

<div></div>

The <div> tag defines a division or a section in an HTML document.
The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
The <div> tag is easily styled by using the class or id attribute.
Any sort of content can be put inside the <div> tag!
Note: By default, browsers always place a line break before and after the <div> element.

dir:

The dir attribute specifies the text direction of the element's content.
It is divided into three types:

1- ltr:

Left-to-right text direction.

2- rtl:

Right-to-left text direction.

3- auto:

Let the browser figure out the text direction, based on the content
(only recommended if the text direction is unknown).

class:

The HTML class attribute is used to specify a class for an HTML element.
Multiple HTML elements can share the same class.
The class attribute is often used to point to a class name in a style sheet.
It can also be used by a JavaScript to access and manipulate elements with the specific class name.

external additives tags


1-

closed tag:

<img>

The <img> tag is used to embed an image in an HTML page.
Images are not technically inserted into a web page; images are linked to web pages.
The <img> tag creates a holding space for the referenced image.

src:

src - Specifies the path to the image.

alt:

alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed.

width:

width - Specifies the width of an image,The unit of measurement is pixels.

height:

height - Specifies the height of an image,The unit of measurement is pixels.


2-

open tag:

<embed ></embed >

The <embed> tag defines a container for an external resource, such as a web page,
a picture, a media player, or a plug-in application.

src:

src - Specifies the address of the external file to embed.

width:

width - Specifies the width of the embedded content,The unit of measurement is pixels.

height:

height - Specifies the height of the embedded content,The unit of measurement is pixels.


3-

open tag:

<video></video>

The <video> tag is used to embed video content in a document, such as a movie clip or other video streams.

src:

src - Specifies the URL of the video file.

width:

width - Specifies the width of an video,The unit of measurement is pixels.

height:

height - Specifies the height of an video,The unit of measurement is pixels.

poster:

poster - Specifies an image to be shown while the video is downloading, or until the user hits the play button.

controls:

controls - Specifies that video controls should be displayed (such as a play/pause button etc).

loop:

loop - Specifies that the video will start over again, every time it is finished.

muted:

muted -Specifies that the audio output of the video should be muted.

autoplay:

autoplay - Specifies that the video will start playing as soon as it is ready.


4-

open tag:

<audio></audio>

The <audio> tag is used to embed sound content in a document, such as music or other audio streams.

src:

src - Specifies the path of the audio file.

controls:

controls - Specifies that audio controls should be displayed (such as a play/pause button etc).

muted:

muted - Specifies that the sound should be muted.

autoplay:

autoplay - Specifies that the audio will start playing as soon as it is ready.

loop:

loop - Specifies that the audio will start over again, every time it is finished.

Signs of tables and lists


1-

open tag:

<ol></ol>

The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.
The <li> tag is used to define each list item.
Tip: Use CSS to style lists.
Tip: For unordered list, use the <ul> tag.

Attributes <ol>:

1: type - Specifies the kind of marker to use in the list,and they: 1/A/a/I/i.
2: start - Specifies the start value of an ordered list (50,51,52...).
3:reversed - Specifies that the list order should be reversed (Z,Y,X...).


2-

open tag:

<ul></ul>

The<ul>tag defines an unordered (bulleted) list.
Use the <ul> tag together with the <li> tag to create unordered lists.
Tip: For ordered lists, use the <ol> tag.
Tip: Use CSS to style lists.


3-

open tag:

<li></li>

The<li> tag is used inside ordered lists<ol>, unordered lists <ul>.
In <ul> the list items will usually be displayed with bullet points.
In <ol>, the list items will usually be displayed with numbers or letters.


4-

open tag:

<table></table>

The<table> tag defines an HTML table.
An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.


5-

open tag:

<tr></tr>

The<tr>tag defines a row in an HTML table.
A <tr> element contains one or more <th> or <td> elements.


6-

open tag:

<th></th>

The<th> tag defines a header cell in an HTML table. An HTML table has two kinds of cells:
Header cells - contains header information (created with the <th> element)
Data cells - contains data (created with the <td> element).


7-

open tag:

<td></td>

The<td>tag defines a standard data cell in an HTML table.
An HTML table has two kinds of cells:
Header cells - (top^)
Data cells - contains data (created with the <td> element)
The text in <td> elements are regular and left-aligned by default.
The text in <th> elements are bold and centered by default.


Attributes <th>and<td>:

1: rowspan - Sets the number of rows a cell should span.
2: colspan - Specifies the number of columns a cell should span.

inputs tags


1-

open tag:

<from></from>

The <form> tag is used to create an HTML form for user input.

action :

The action attribute defines the action to be performed when the form is submitted.
Usually, the form data is sent to a file on the server when the user clicks on the submit button.

2-

closed tag:

<input>

The <input> tag specifies an input field where the user can enter data.
The <input> element is the most important form element.
The <input> element can be displayed in several ways, depending on the type attribute.

The different input types are as follows:

( <input type="button">/ <input type="checkbox">/ <input type="date">/ <input type="email">/ <input type="file">/
<input type="month">/ <input type="number">/ <input type="password">/ <input type="radio">/
<input type="range">/ <input type="reset">/ <input type="search">/ <input type="submit">/
<input type="tel">/ <input type="text">/ <input type="time">/ <input type="url">...........).

placeholder :

Specifies a short hint describing the expected value or address of the input input element.

name :

Definition and UsageThe name attribute specifies the name of an <input> element.
The name attribute is used to reference elements in a JavaScript, or to reference form data after a form is submitted.

value :

The value attribute specifies the value of an <input> element.
The value attribute is used differently for different input types:

For "button", "reset", and "submit" - it defines the text on the button.
For "text", "password", and "hidden" - it defines the initial (default) value of the input field.
For "checkbox", "radio", "image" - it defines the value associated with the input (this is also the value that is sent on submit).

Note: The value attribute cannot be used with <input type="file">.

3-

tag open:

<label></label>

Users who have difficulty clicking on very small regions (such as radio):
because when a user clicks the text within the <label> element, it toggles the input (this increases the hit area).

Tip: The for attribute of <label> must be equal to the id attribute of the related element to bind them together.

for :

Specifies the id of the form element the label should be bound to

4-

open tag:

<textarea></textarea>

The <textarea> tag defines a multi-line text input control.
The <textarea> element is often used in a form, to collect user inputs like comments or reviews.
The name attribute is needed to reference the form data after the form is submitted
(if you omit the name attribute, no data from the text area will be submitted, More important tip than with PHP ).
The id attribute is needed to associate the text area with a label .
Tip: Always add the <label> tag for best accessibility practices!

5-

open tag:

<select></select>

The <select> element is used to create a drop-down list.
The <select> element is most often used in a form, to collect user input.
The name attribute is needed to reference the form data after the form is submitted
(if you omit the name attribute, no data from the drop-down list will be submitted, More important tip than with PHP ).
The id attribute is needed to associate the drop-down list with a label .
The <option> tags inside the <select> element define the available options in the drop-down list.
Tip: Always add the <label> tag for best accessibility practices!

6-

open tag:

<option></option>

The <option> tag defines an option in a select list.
Note: The <option> tag can be used without any attributes, but you usually need the value attribute,
which indicates what is sent to the server on form submission.
